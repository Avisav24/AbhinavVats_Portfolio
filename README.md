# Abhinav Vats Portfolio

A premium, interactive 3D portfolio built with **Next.js**, **Framer Motion**, and **Tailwind CSS**. Features a deep-space aesthetic with dark glassmorphism, ambient audio, and mobile-responsive design.

## 🚀 Features

- **Deep-Space Aesthetic**: Dark liquid-glass components with frosted blur effects.
- **Ambient Experience**: Integrated sound manager with background music and hover SFX.
- **Mobile Optimized**: Fully responsive layout across all devices.
- **Tech Stack**:
  - **Framework**: Next.js 14
  - **Animations**: Framer Motion
  - **Icons**: Lucide React & Simple Icons
  - **Forms**: Integrated with Web3Forms
  - **Styling**: Tailwind CSS & Glassmorphism

## 🛠️ Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/Avisav24/AbhinavVats_Portfolio.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```
